# Product Overview

Lightweight ticketing/helpdesk system for managing customer support tickets with role-based access control.

## Core Features

- **User Management**: Three roles (Admin, Agent, Customer) with distinct permissions
- **Product Management**: Admin can manage product catalog for ticket categorization
- **Ticket Lifecycle**: Create, assign, track, and resolve support tickets with product association
- **Status Tracking**: Open → In Progress → Resolved → Closed
- **Priority Levels**: Low, Medium, High, Urgent
- **Comments System**: Internal notes (staff only) and public comments
- **Dashboard**: Statistics including ticket status, top customers by ticket count, and products with most issues

## User Roles & Permissions

- **Admin**: Full system access, user management, product management, all tickets
- **Agent**: View all tickets, manage assigned tickets, add internal notes
- **Customer**: Create tickets with product selection, view own tickets only, add public comments
